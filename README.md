# ChessKid Super Arena

This repository now ships a **much more ChessKid-like** single-page experience with:

- Accounts (kid/parent roles), profile cards, badges/currency
- Quick match, private room codes, and bot opponents
- Safe in-game chat
- Lessons tab with completion rewards
- Daily puzzle tab
- Clubs directory
- Parent controls panel
- Chess rules including legal move generation with castling, en passant, check/checkmate/stalemate detection

## Run

```bash
node server.js
```

Then open `http://localhost:3000`.

## Important note

This is still an independent, from-scratch project and not affiliated with ChessKid. It reproduces a similar structure and feature set for educational/demo use, but it is not an exact clone of ChessKid production systems.
